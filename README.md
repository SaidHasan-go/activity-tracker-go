# Activity Tracker (Go)

Backend-проект на Go — трекер дневной активности и тренировок.

Проект реализует обработку данных о шагах и тренировках, расчёт дистанции, средней скорости и потраченных калорий. Архитектура построена с использованием интерфейсов и разделения ответственности.

## Возможности
- Учёт дневной активности (шаги, длительность)
- Учёт тренировок (ходьба, бег)
- Расчёт:
  - дистанции
  - средней скорости
  - потраченных калорий
- Валидация входных данных
- Расширяемая архитектура через интерфейсы

## Стек технологий
- Go
- Стандартная библиотека
- Интерфейсы
- Работа со временем (`time.Duration`)
- Структурирование кода по пакетам

## Структура проекта
internal/
├── actioninfo // обработка и вывод информации
├── daysteps // дневная активность
├── trainings // тренировки
├── spentenergy // расчёт калорий и дистанции
└── personaldata // данные пользователя
